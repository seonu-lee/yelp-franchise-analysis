## 1. STEP 5 — FPI 산출

### 분석 방법
- **FPI 공식**: FPI = Σ { (별점 × ln(리뷰수+1) × 감성점수_shifted) / (거리(km)+k)² }
- **Shift 정규화 적용**: 프랜차이즈 감성점수의 음수값 문제 해결
  - 기존: 음수 감성점수 → FPI 음수 → 1,292개 NaN 발생
  - 수정: sentiment_adj = sentiment - min(sentiment) → FPI 항상 0 이상
- **임계거리**: 300m (민감도 분석 결과 별점+감성점수 모두 p<0.05인 최소 반경)
- **보정 상수**: k=1 (거리 km 단위, 극단값 발산 방지)

### 민감도 분석 결과

| 반경 | 별점 p-value | 감성점수 p-value |
|---|---|---|
| 100m | 0.0001 ✅ | 0.2672 ❌ |
| 200m | 0.0002 ✅ | 0.5950 ❌ |
| **300m** | **0.0000 ✅** | **0.0207 ✅** |
| 500m | 0.0000 ✅ | 0.0017 ✅ |

### FPI 구간 분류

| 구간 | 정의 | 업체 수 | 비율 |
|---|---|---|---|
| NP (무풍지대) | fpi_300m = 0 | 977개 | 20.3% |
| LP (저압력) | fpi_300m > 0 하위 50% | 1,921개 | 39.9% |
| HP (고압력) | fpi_300m > 0 상위 50% | 1,920개 | 39.9% |

### 설계적 한계
FPI 산출에 프랜차이즈 리뷰수를 사용하고, 통제변수로 독립 브랜드 리뷰수를 사용하여
간접적 상관관계(r=0.074)가 존재할 수 있다. FPI 효과가 일부 과소추정될 가능성을 배제할 수 없다.

## 2. STEP 6a — FPI 영향 분석 (회귀분석 + 구간별 성과 비교)

### 분석 조건
- 분석 대상: 영업 중인 독립 브랜드 (is_open=1), 3,017개
- 통제변수: log(review_count), neighborhood 더미 (Unknown 처리)
- 회귀 방법: OLS + HC3 Robust Standard Error

### 다중회귀분석 결과

| 종속변수 | FPI 계수 | P-value | 유의미 여부 |
|---|---|---|---|
| 별점 | -0.0119 | 0.033 | ✅ p<0.05 |
| 감성점수 | -0.0001 | 0.384 | ❌ p≥0.05 |

- FPI가 높을수록 독립 브랜드 별점이 유의미하게 낮아짐
- 감성점수는 FPI보다 매장 규모·상권 특성에 더 민감하게 반응
- HC3 적용 후 계수(-0.0119)와 유의성(p=0.033) 동일하게 유지 → 결과 견고성 확인

### ANOVA + Tukey 결과

**별점 (p=0.000 ✅)**

| 구간 | 평균 별점 | Tukey 결과 |
|---|---|---|
| NP | 3.83 | NP > LP, NP > HP, LP > HP 모두 유의미 |
| LP | 3.69 | |
| HP | 3.61 | |

**감성점수 (p=0.023 ✅)**

| 구간 | 평균 감성점수 | Tukey 결과 |
|---|---|---|
| NP | 0.0133 | HP vs NP만 유의미 |
| LP | 0.0124 | |
| HP | 0.0115 | |

### 해석
ANOVA에서 통제변수 없이 단순 집단 비교 시 FPI 효과가 착시에 가려졌으나,
다중회귀에서 외부 요인을 통제하면 FPI가 별점에 음(-)의 유의미한 영향을 미치는 것이 확인됨.

## 3. STEP 6b — TF-IDF 단어 → 별점 회귀분석

### 분석 조건
- 분석 대상: 영업 중인 독립 브랜드 3,017개
- 독립변수: 브랜드별 TF-IDF (min_df=0.05, max_df=0.90, max_features=500)
- 전처리: log(x+1) + StandardScaler
- 회귀 방법: OLS + HC3
- VIF 최대값: 5.87 (다중공선성 없음)

### 모델 성능

| 항목 | 값 |
|---|---|
| R² | 0.8227 |
| Adj R² | 0.7874 |
| F-statistic | 22.80 (p=0.000) |
| 유의미한 단어 | 40개 (긍정 17개, 부정 23개) |

### 주요 결과

**별점 상승 단어**: awesom(+0.065), amaz(+0.056), delici(+0.046), favorit(+0.042), excel(+0.035)

**별점 하락 단어**: worst(-0.061), money(-0.061), rude(-0.059), slow(-0.048), horribl(-0.047)

### 해석
별점을 결정하는 핵심 요인은 메뉴 종류가 아니라
**고객의 감정·평가 표현(awesom, perfect, worst, rude)**임이 확인되었다.

## 6. 종합 결론

### 연구질문별 답변

**Q1. FPI가 높을수록 독립 브랜드 별점은 어떻게 달라지는가?**
> FPI가 높을수록 독립 브랜드 별점이 유의미하게 낮아진다 (coef=-0.012, p=0.033).
> 구간별로는 NP(3.83) > LP(3.69) > HP(3.61) 순으로 하향 패턴이 나타났다.

**Q2. 임계거리는 얼마인가?**
> 별점과 감성점수 두 종속변수 모두 p<0.05인 최소 반경은 **300m**이다.
> 이는 라스베가스 도심 상권에서 소비자가 실제로 비교 탐색하는 현실적 거리와도 부합한다.

**Q3. 고압력 환경에서 살아남는 독립 브랜드의 차별화 전략은?**
> 음식 품질(delici, amaz, awesom)과 긍정적 경험(favorit, perfect)으로 차별화한다.
> 서비스 불만(bad, minut, manag)을 최소화하는 것이 생존의 핵심이다.
> 프랜차이즈가 잘하는 것(빠른 서비스, 표준화)을 따라하는 것이 아니라
> 프랜차이즈가 못하는 것(전문 음식, 개인화된 서비스 경험)으로 차별화해야 한다.

**Q4. FPI 구간별로 독립 브랜드 리뷰에서 추출되는 키워드가 달라지는가?**
> 구간별로 고유 키워드가 뚜렷하게 달라지는 패턴이 확인되었다.
>
> - NP(무풍지대): thai, pho, coffee, buffet, noodle → 에스닉 전문점, 특색 음식 카테고리
> - LP(저압력): pizza, steak, sushi, ramen, korean → 정통 외국 요리 전문성
> - HP(고압력): burger, drink, cheese, waiter, server → 캐주얼 다이닝 + 서비스 언급
>
> FPI가 높아질수록 독립 브랜드 메뉴가 프랜차이즈와 유사해지는 경향이 나타나며,
> 고압력 환경에서는 메뉴 차별화보다 **서비스 경험**이 리뷰의 주요 화제로 부상한다.
> 이는 소비자가 프랜차이즈 밀집 지역에서 독립 브랜드를 평가할 때
> 메뉴보다 서비스 품질에 더 주목함을 시사한다.

### 설계적 한계
- Restaurants 카테고리 전체 분석으로 업종 이질성 존재
- FPI-리뷰수 간접 상관(r=0.074) → FPI 효과 일부 과소추정 가능성

- 브랜드명 정제 한계로 일부 프랜차이즈 누락 가능성
